# Snake Client Project

Snake is a popular video game that I discovered on my cousin's Nokia phone in 2008. Since its creation in 1997, it has taken on multiple forms. 

This is a barebones version of the game, with some multiplayer features added. 

Before you can run this client, you will need to be running the server side which you can download and install from [here.](https://github.com/lighthouse-labs/snek-multiplayer)

## Getting Started

- Follow steps [inside the snek server repo](https://github.com/lighthouse-labs/snek-multiplayer) to run the server side
- Clone the snake client, and run using the following commands. 
```
git clone https://github.com/ItsAhmad/snake-client.git
cd snake-client
node play.js

```


## How to play 
- Guide your snake to eat the red dot. 
- With every red dot you consume, the snake gets longer, you will find it increasingly more difficult to manuever the snake.
- Avoid hitting the walls or the body of the snake. 


## Controls 
- `W` `A` `S` `D` - Movement
- `1` - "I'm gonna win!"
- `2` - "I gotta catch up!"
- `Ctrl` + `C `- EXIT GAME
